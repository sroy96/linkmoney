# Fintech Pitch Deck

A premium, dark-mode fintech pitch deck built with React + TypeScript + Vite + Tailwind CSS.

## Slides
1. **Revenue Per User** — ₹3,400 blended revenue across credit lifecycle
2. **TAM / SAM / SOM** — Market opportunity in credit improvement & loan optimisation

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Stack
- React 18 + TypeScript
- Vite 5
- Tailwind CSS 3
- Lucide React icons
- Google Fonts: Syne (display) + DM Sans (body)
